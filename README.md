# 📌 Personal Expense Tracker (CLI)

A command-line based expense management tool built with Python that helps users record, organize, and analyze their daily spending.

This project focuses on practical financial tracking through a simple yet powerful CLI interface.

---

## 🚀 Features

- Add expenses by category  
- Delete existing expenses  
- View all expenses  
- View expenses by specific category  
- Identify highest spending category  
- View expenses sorted by amount  
- Total spending calculation  
- Average spending analysis  
- Persistent storage using CSV  
- Input validation for safe usage  

---

## 🛠 Tech Stack

- Python  
- CSV File Handling  
- Command-Line Interface (CLI)

## 📊 Functionalities

| Operation | Description |
|----------|-------------|
| Add Expense | Add new category or update existing |
| View Expenses | Displays all expenses with totals |
| Delete Expense | Removes a category |
| Category View | View specific category spending |
| Highest Expense | Shows max spending category |
| Sorted View | Lists expenses by amount |
| Analytics | Displays total & average spending |

---

## 💾 Data Storage

All expenses are stored locally using a CSV file (`expense.csv`) which is automatically created when the program runs.
